# 大明王朝：开局一个碗 (Ming Dynasty RPG: Starting with a Bowl)

大明王朝：开局一个碗 是一款基于 Pygame 开发的 8-bit NES 复古风格像素角色扮演游戏（RPG）。玩家将扮演少年朱元璋（朱重八），从一个食不果腹、手持破碗的行童开始，在乱世中结识徐达、常遇春等开国功臣，反抗元朝暴政，最终攻克元大都，推翻元朝统治，建立大明王朝。

---

## 游戏背景

公元 1352 年（至正十二年），元朝末年，朝廷腐败，贪官污吏横行，百姓民不聊生。加之黄河决口，瘟疫与饥荒席卷中原，神州大地满目疮痍。

在濠州府钟离县，一个名叫朱重八的少年在几天内失去了所有的亲人。为了活命，他削发为僧，进入皇觉寺。然而，乱世之中佛门亦难避兵燹之灾，寺庙无粮，主持只得让行童们下山化缘。

手持一口行军破碗，朱重八踏上了他的化缘之路。他不会想到，这口用来乞讨的破碗，将装下整片锦绣江山；而他的脚下，正是一条通往九五之尊、开创三百年大明盛世的帝王之路……

## 游戏特色

### 1. 纯正 8-bit NES 复古美学
* **复古像素视觉**：游戏采用经典的 8-bit 像素美术风格，所有角色、建筑、地形均经过精心设计，还原红白机时代的视觉感动。
* **平滑网格移动**：支持流畅的四方向帧动画行走，结合平滑的滑动过渡，带来极佳的操作手感。
* **复古音效生成**：内置基于实时正弦波/方波合成的复古音效系统，无需外部音频文件即可播放经典的交互、成功和战斗音效。

### 2. 动态敌人与智能 AI 系统
* **智能寻路追击**：元军士兵与场景 Boss 拥有巡逻（Wander）与追击（Chase）双重 AI 状态。一旦玩家进入警戒范围，敌人将采用曼哈顿距离算法实时寻路包抄玩家。
* **场景战力自适应**：敌人的生命值、攻击力、防御力与经验奖励完全由场景的推荐等级决定，随着剧情推进，敌军战力呈线性增长，极具挑战性。
* **历史名将 Boss**：每个场景深处均驻守着对应的元朝历史名将（如哈剌不花、答失八都鲁、也先帖木儿等）。Boss 拥有更庞大的像素体型（1.5x - 2.0x）与专属的冠冕标识。

### 3. 独创“砍树爆装”与 10 大装备槽位
* **场景互动探索**：靠近场景中的树木即可自动进行砍伐。砍伐完成后将随机掉落各种品质的装备。
* **全方位装备系统**：拥有武器、护腕、项链、护符、戒指、帽子、护手、护肩、腰带、靴子等 10 大独立装备槽位。
* **智能装备对比**：掉落装备时，系统会自动对比当前槽位装备的属性，并提供直观的对比界面，玩家可通过快捷键一键替换或丢弃。

### 4. 挂机修行（AFK）系统
* **离线/闲置收益**：完美融合现代放置类游戏机制，按下快捷键即可开启/关闭“挂机修行”模式。在安全区域闲置时，主角将持续获得修行经验，助你轻松度过前期难关。

### 5. 宏大的历史战役与多场景切换
* **历史场景还原**：从皇觉寺起兵，历经淮西、河南、驴牌寨、和州、太平、应天、鄱阳湖决战，直至攻克元大都。
* **大结局剧情**：击败镇守元大都的终极 Boss 元顺帝，即可触发波澜壮阔的“大明开国”大结局，登基称帝，改元洪武。

## 操作指南

| 按键 | 功能说明 |
| :--- | :--- |
| **W / A / S / D** 或 **方向键** | 控制主角上下左右移动 |
| **Space (空格键)** | 与 NPC 对话、打开宝箱、推进对话文本 |
| **G** | 开启/关闭“挂机修行（AFK）”模式（每秒获得经验） |
| **1 / Y** | 在装备掉落对比界面选择【装备/替换】 |
| **2 / N / Esc / Space** | 在装备掉落对比界面选择【丢弃】 |
| **Enter (回车键)** | 推进大段历史剧情背景文本 |

## 快速开始

### 环境依赖
游戏运行需要 Python 3.8 或更高版本，并安装以下第三方库：
* **Pygame**：用于游戏引擎渲染、事件处理与音效播放。
* **Pillow (PIL)**：用于在游戏首次启动时动态生成角色与敌人的走路帧动画。

### 安装步骤

1. 克隆本项目到本地：
   ```bash
   git clone https://github.com/customercarelinda-bot/MingDynastyGame.git
   cd MingDynastyGame
   ```

2. 安装依赖库：
   ```bash
   pip install pygame pillow
   ```

3. 运行游戏：
   ```bash
   python main.py
   ```

## 项目结构

```text
MingDynastyGame/
├── main.py                  # 游戏主引擎与核心逻辑
├── map_data.py              # 静态与动态场景地图数据加载器
├── enemy_renderer.py        # 敌人与 Boss 的独立渲染模块
├── map_editor.py            # 地图编辑器（用于设计关卡）
├── assets/                  # 游戏资源目录
│   ├── tiles/               # 地形与障碍物像素贴图
│   ├── player/              # 动态生成的玩家帧动画
│   ├── enemies/             # 动态生成的敌人与 Boss 帧动画
│   ├── npcs/                # 动态生成的 NPC 贴图
│   ├── equipment_table.json # 装备属性数据库
│   └── monster_table.json   # 怪物属性数据库
└── tools/                   # 资源自动生成工具集
    ├── generate_player.py   # 主角帧动画生成脚本
    ├── generate_enemies.py  # 敌人与 Boss 帧动画生成脚本
    └── generate_npcs.py     # 基础 NPC 贴图生成脚本
```
